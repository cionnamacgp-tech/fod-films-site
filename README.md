# Fód Films — Website

This is a Vite + React + Tailwind starter with your brand and contact form wired to Formspree (`/f/xdkprbln`).

## Run locally
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## Deploy (public)
1. Create a free account at https://vercel.com (or Netlify).
2. New Project → **Import** this folder (or drag & drop).
3. Build command: `vite build` (auto)
4. Output dir: `dist` (auto)
5. Deploy — you'll get a public URL.

## Swap emails later
When info@fodfilms.ie and ken@fodfilms.ie are live, update the Formspree destination emails in your Formspree dashboard.
